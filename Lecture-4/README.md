# Lecture-4 JavaScript进阶

负责人：吴思

## 课程目的

1. 复习JavaScript基础，在上次培训的基础上进一步学习JavaScript。

## 教学方法

1. 结合ppt和demo演示，以讲为主。
2. 布置课后作业。


## 课程结构和内容

1. 复习javascript基础
    - javascript简介
    - 数据类型
    - 基本语法、函数
    - DOM
    
2. 事件
    - 什么是事件
    - 事件流
    - 事件处理程序
    - 事件类型
    - Event对象

3. 对象
    - 创建对象的不同方法与区别
        - 字面量方法
        - 构造函数模式
        - 原型模式
        - 原型模式与构造函数相结合
    - 继承
        - 理解继承
        - 继承的实现与分析
            - 通过原型链继承
            - 借助构造函数和原型链组合继承
            - 其他方法及注意事项

4. 作用域与闭包
    - javascript作用域
    - 什么是闭包
    - 理解闭包函数创建、调用的整个过程
    - 闭包的使用场合及优缺点
    - 常见的闭包问题举例：
        - this指向
        - 循环中的闭包问题
